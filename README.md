# pandas-employee-data-analysis
Pandas Data Selection and Filtering Assignment using Employee Dataset.

->This project demonstrates how to use **Pandas selection and filtering techniques** to analyze employee data.

->The dataset contains information about employees such as EmployeeID, Name, Department, Salary, Performance Score, and Years of Service.

#Tasks Performed:
1. Created a Pandas DataFrame using the provided employee dataset.
2. Selected specific columns (Name, Department, Salary).
3. Used **iloc** to access data using row position.
4. Used **loc** to access data using EmployeeID.
5. Filtered employees from the Engineering department.
6. Filtered employees with salary greater than 70,000.
7. Filtered Sales employees with Performance Score ≥ 4.
8. Sorted employees by salary and displayed the top 3 highest-paid employees.
9. Sorted data by Department and Salary within each department.

#Tools Used:
- Python
- Pandas
- Google Colab
